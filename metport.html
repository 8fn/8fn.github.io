005 : MetPort

Python project built with Flask and Folium that scrapes data with BS4 to display the temperature of cities in Portugal. Folium is used to create interactive maps. Check my project here

The project was built using Python 3 and the Flask framework, and was deployed using Heroku. The main feature of the website is a map created using Folium, which displays the weather and temperature data for cities in Portugal. To obtain this data, the website uses Beautiful Soup (bs4) to scrape weather information from various sources. Overall, the website provides a user-friendly interface for exploring weather conditions in Portugal, with up-to-date information that is both accurate and reliable.
